Quick how-to guide!

Some commands you can use

  • /name <name> - Changes your name from Stranger### to whatever you want.
  • /disconnect - For if you can't handle it anymore.
  • ...suggest something!

Some basic ideas

  • Yes, this is a real chatgroup.
  • Yes, you are talking to real people.
  • Names show up as < John> for regular users
  • Names show up as <@Bob> for moderators
  • Names show up as <~Angelo> for admins

Technical details

This project is made possible by combining Omegle's stranger system with IRC channels.
